Design systems from a single base color

Rampa CLI uses mathematical distributions and harmony theory to generate complete palettes you can trust across UI, data visualization, and brand accents.

Palette Generation

Generate 2-100 colors from any base color. Control lightness, saturation, and hue ranges with mathematical distribution curves (linear, fibonacci, golden-ratio, logarithmic, and more).

Color Harmonies

Add complementary, triadic, analogous, split-complementary, square, or custom hue-shifted ramps. Build complete design systems from a single color.

Accessibility Built-in

APCA contrast validation for every color pair. Filter by use case: body text, large text, bold labels, non-text elements. Never ship inaccessible color combinations again.

Multiple Outputs

Export as plain text, JSON, or CSS custom properties. Pipe into your build pipeline or paste directly into your codebase.
