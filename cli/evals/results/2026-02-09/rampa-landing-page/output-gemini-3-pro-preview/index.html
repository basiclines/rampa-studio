Palette Generation

Generate 2-100 colors from any base color. Control lightness, saturation, and hue ranges with mathematical distribution curves.

Color Harmonies

Add complementary, triadic, analogous, or custom hue-shifted ramps. Build complete design systems from a single color.

Accessibility Built-in

APCA contrast validation for every color pair. Filter by use case: body text, large text, UI elements.

Multiple Outputs

Export as plain text, JSON, or CSS custom properties. Pipe directly into your build pipeline.

Designed for the CLI

Rampa follows the Unix philosophy. It does one thing well: generating colors. Pipe the output, script it, or copy-paste it.

1

Simple defaults

Just provide a base color. Rampa handles the rest.

2

Advanced control

Fine-tune lightness curves, saturation decay, and hue shifting.

Safety first

The -A flag ensures you never ship inaccessible contrast.

rampa — -zsh
