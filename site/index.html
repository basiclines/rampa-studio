Define ranges for lightness, saturation, and hue. Pick a distribution scale. Get a mathematically consistent palette with contrast analysis included.

Range-based control

Configure lightness, saturation, and hue shift as start:end ranges. Choose from 11 distribution scales — linear, fibonacci, golden-ratio, ease curves, and more. Export as CSS custom properties, JSON, or plain text. 

rampa -C "#3b82f6" -L 95:10 --lightness-scale=fibonacci -O css

Color harmonies, derived automatically

Generate complementary, triadic, analogous, split-complementary, square, and compound ramps from a single base color. Add custom hue shifts with shift:N. 

rampa -C "#3b82f6" --add=complementary --add=analogous -O css

APCA contrast analysis, built in

Analyze all color pairs across every ramp. Filter by use case — body text, large text, non-text elements. Output as text, JSON, or CSS comments. 

rampa -C "#3b82f6" --add=complementary -A=body -O json

Tinting with 16 blend modes

Apply any color as an overlay with adjustable opacity. Multiply, overlay, screen, and 13 more. Useful for brand-tinted neutrals. 

rampa -C "#6B7280" --tint-color="#3b82f6" --tint-opacity=15 --tint-blend=overlay

Color conversion, structured output

Read a color in any format and convert it to hex, rgb, hsl, or oklch. Use --read-only to skip ramp generation. JSON output returns structured values — no CSS strings to parse. 

rampa -C "#3b82f6" --read-only -O json -F hsl
{ "color": { "value": "hsl(217, 91%, 60%)", "hsl": { "h": 217, "s": 91, "l": 60 } } }

JavaScript SDK

Same color engine, programmatic API. Use rampa in your applications, build tools, and design systems without the CLI. Fluent, chainable interface — every CLI flag maps to a method.

CLI ↔ SDK

Every CLI command has a 1:1 SDK equivalent. Same engine, same output.

Basic palette

CLI

rampa -C "#3b82f6" --size=5

SDK

rampa('#3b82f6').size(5).generate()

Ranges and scales

CLI

rampa -C "#3b82f6" -L 10:90 \
  -S 80:20 --lightness-scale=fibonacci

SDK

rampa('#3b82f6')
  .lightness(10, 90)
  .saturation(80, 20)
  .lightnessScale('fibonacci')
  .generate()

Harmonies

CLI

rampa -C "#3b82f6" \
  --add=complementary \
  --add=shift:45

SDK

rampa('#3b82f6')
  .add('complementary')
  .add('shift', 45)
  .generate()

Tinting

CLI

rampa -C "#6B7280" \
  --tint-color="#3b82f6" \
  --tint-opacity=15 \
  --tint-blend=overlay

SDK

rampa('#6B7280')
  .tint('#3b82f6', 15, 'overlay')
  .generate()

CSS output

CLI

rampa -C "#3b82f6" -O css

SDK

rampa('#3b82f6').toCSS()

Color conversion (read-only)

CLI

rampa -C "#fe0000" --read-only
rampa -C "#fe0000" --read-only -F hsl

SDK

rampa.readOnly('#fe0000').generate()
rampa.readOnly('#fe0000').format('hsl').generate()

Output format

CLI

rampa -C "#3b82f6" -F oklch

SDK

rampa('#3b82f6').format('oklch').generate()

Feature reference

Color Spaces OKLCH-based engine. Accepts hex, hsl, rgb, oklch input.
Ranges Lightness, saturation, hue shift — all configurable as start:end.
Scales 11 types: linear, fibonacci, golden-ratio, ease-in-out, geometric, logarithmic, and more.
Harmonies 6 harmony types + custom shift:N degrees.
Tinting 16 blend modes, adjustable opacity.
Accessibility APCA contrast analysis, 6 filter levels, JSON/CSS/text output.
Output CSS custom properties, JSON (structured values), plain text.
Conversion Read-only mode for single color conversion between formats.
Platforms macOS (ARM, Intel), Linux (x64, ARM64), Windows. npm, Homebrew, standalone binary.
SDK Fluent JavaScript/TypeScript API. Same engine as the CLI. npm install @basiclines/rampa-sdk
