Features

Everything you need for
production color systems

One tool generates palettes, analyzes contrast, mixes colors, and exports to any format. No plugins, no dependencies.

Range-based control

Configure lightness, saturation, and hue shift as start:end ranges. Choose from 11 distribution scales — linear, fibonacci, golden-ratio, ease curves.

Color harmonies

Generate complementary, triadic, analogous, split-complementary, square, and compound ramps. Custom hue shifts with shift:N.

APCA contrast

Analyze all color pairs. Filter by body text, large text, non-text. Built from the W3C spec — no external dependencies.

16 blend modes

Apply any color as an overlay with adjustable opacity. Multiply, overlay, screen — perfect for brand-tinted neutrals.

Color spaces

LinearColorSpace for 2-color ramps. CubeColorSpace for 8-corner cubes. OKLCH, LAB, or RGB interpolation.

Color conversion & structured output

Read any color format. Convert to hex, rgb, hsl, or oklch. JSON returns structured values — no CSS strings to parse. CSS output generates ready-to-use custom properties.

Contrast lint

APCA or WCAG 2.x between any two colors. Pass/fail per level. Warnings for near-identical colors and pure black/white.
